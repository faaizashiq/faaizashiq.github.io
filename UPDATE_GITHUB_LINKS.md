# Update GitHub Repository Links

## How to Update Project GitHub Links

1. Open `script.js` file
2. Find the `projectsData` object (around line 150)
3. Update the `githubUrl` property for each project with your actual GitHub repository URLs

### Example:
```javascript
1: {
    // ... other properties
    githubUrl: "https://github.com/yourusername/image-classification-model"  // ← Update this
},
```

### Replace with your actual GitHub URLs:
- Project 1: Image Classification Model
- Project 2: NLP Sentiment Analysis  
- Project 3: Predictive Analytics Model
- Project 4: Computer Vision App
- Project 5: Recommendation System
- Project 6: ML-Powered Mobile App

## Current Placeholder URLs:
All projects currently use placeholder URLs like:
`https://github.com/yourusername/project-name`

Replace `yourusername` with your actual GitHub username and `project-name` with your actual repository names.

## Testing:
After updating, click on any project card to see the modal with details and the GitHub link at the bottom.
